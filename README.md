# 🌐 Holberton Web Back-End

This repository contains projects from the **Holberton School Web Back-End curriculum**.
The focus of these projects is learning how back-end systems work using **Python**, databases, and APIs.

The repository includes exercises related to **data storage, API development, and server-side logic**.

---

# 📚 Project Overview

The projects in this repository explore important back-end concepts such as:

* server-side development
* database interaction
* data persistence
* API communication
* working with structured and unstructured data

---

# 🐍 Technologies Used

* **Python 3**
* **MongoDB**
* **NoSQL databases**
* **REST APIs**
* **JSON**
* **Linux / Command Line**
